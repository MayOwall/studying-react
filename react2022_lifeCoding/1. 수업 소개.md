### 리액트의 핵심적인 역할

=> 사용자 정의 태그를 만드는 것

### 리액트를 만드는 방법 2가지

1. class 사용 <br/>
   전통적인 방식.
2. function 사용 <br/>
   최근에는 함수 방식으로 react를 사용하는 것이 늘어남.
